# PROYECTO PORTAFOLIO DE EVELYN
---
## 📋 Proceso de Desarrollo
Para construir este portafolio, seguí un flujo de trabajo organizado:
1. **Estructuración Semántica:** Definición de la arquitectura del sitio usando etiquetas HTML5 ademas de ordenar el css para que sea entendible
2. **Diseño Visual:** Creación de una identidad visual basada en tonos celestes y azules usando variables CSS.
3. **Responsive Design:** Adaptación de la interfaz para dispositivos móviles y escritorio mediante Media Queries.
4. **Interactividad:** Implementación de efectos visuales (hover, animaciones fade-up) para mejorar la experiencia del usuario.

## 🛠️ Tecnologías y Metodologías
Para este proyecto utilicé herramientas y flujos de trabajo :

* **Tecnologías:** HTML5 y CSS3 (enfocado en Flexbox y Grid).
* **Metodologías Ágiles:** Organización de tareas por notion
* **Control de Versiones:** Uso de **Git** y **GitHub** siguiendo la convención de commits para documentar el progreso.
* **Enfoque UI/UX:** Jerarquía visual clara y navegación intuitiva.
